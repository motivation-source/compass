One-time setup

Add your Groq API key

The chat mode uses Groq's free Llama API to power conversations. You'll need a key, which you get for free.

  1. Go to console.groq.com and sign in (free, no credit card)
  2. In the left sidebar, click API Keys
  3. Click Create API Key, name it anything, and submit
  4. Copy the key (starts with gsk_) — it's only shown once
  5. Paste it below

Your key is stored only in this browser, in localStorage. It's never sent anywhere except to Groq when you chat. You can remove or change it any time from the ⚙ menu.
